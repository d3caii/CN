# CN
##Compilation
    type `make`
##Running the server
    `./server <port>`
    The server will start on the system 0.0.0.0:<port>
###Exiting
    `quit_server`
##Connecting to the server(Running the Client)
    `./client <server ip> <port>`
    You will need to kow the ip of the machine hosting the server as well as the port
###Exiting
    `quit`
